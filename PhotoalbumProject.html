To create album: Enter Album name in textbox, hover over "menu", click on "create new album". To upload media to album: Select album that is to be uploaded to, hover over "menu", click on "upload", and select file that is to be uploaded. To display media from album: Select album that is to be displayed, click on "open selected album".
